Knobs 'n' Dials - QML
=========

![ScreenShot](https://raw.github.com/ashish157/Knobs-n-Dials-QML/master/Screenshot.png)

Qt and QML based different types of Knobs and Dials

To build it you need a Qt >= 5.1.
To compile, just run qmake && make.
